## 1. Describe your process. At a high level, how did you go about creating this project?

After getting the idea I planned it out using Google Gemini. I structured 
the project into setting up the website, then went one game at a time and 
finished with a strong UI polish and editing code for efficiency.

## 2. What AI tools and strategies did you use? What models did you choose, and were you using them in a browser chat, in something like Cursor, in the command line, etc? 

I used Gemini Pro 3.1 in borwser chat for planning next moves, the 
project structure and to see how feasible each idea I had is. I also 
used Cursor, mainly in the ask and agent mode. I asked questions about 
the code in ask and auto mode to get quick efficient answers and I
generated code in Opus4.7 agentic mode to get the best product.

## 3. Did you use an agentic process like in HW8, or something else?

It was similar, but instead of keeping track of many SPEC.md and 
REVIEW.md files I organized my chats to cover specific parts of
the website. Iniside the chats I wrote the plan like in HW8 and 
finished with the agent putting the crucial prompts into PROMPT.md 
and another agent to execute a thorough review at every checkpoint.

## 4. Why did you make the choices above?

There were many files and parts to keep track of. It seemed too
much to work on anything at once, especially since I didn't have
vision of how features would actually be implemented.
